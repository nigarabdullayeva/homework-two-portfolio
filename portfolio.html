Trivia-Night Project

As a team we decided to create a quiz game application. Trivia Night is a fun trivia game with persistent profile stat tracking and top scores.It uses Open Trivia DB API and generates a quiz game. Users are asked to sign up or log in, then the app displays the user's profile and allows for start of the game. The quiz consists of ten questions from different categories and difficulty types. User has 90 seconds to answer the questions and get personal stats.By clicking HERE you will be forwarding to the GitHub repository.
